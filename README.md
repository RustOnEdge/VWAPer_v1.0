# VWAPer v0.1

A simple C++ program to process stock market data, calculate volume percentages per interval, and track daily highs/lows.

## Features

- Reads market data: `[Stock] [Interval] [Volume] [High] [Low]`
- Calculates total and interval-specific volume per stock
- Outputs `% volume per interval` and daily high/low
- Uses `#` as delimiter between outputs
